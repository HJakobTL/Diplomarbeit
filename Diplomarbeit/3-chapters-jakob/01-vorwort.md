# Vorwort

Daten sind das Fundament jeder Anwendung -- das ist eine Binsenweisheit,
aber erst, wenn man selbst eine Datenbank entwirft, versteht man, wie viel
dahintersteckt. Mich hat schon immer die Frage beschäftigt, wie man
Informationen so strukturiert, dass sie widerspruchsfrei, dauerhaft und
schnell abrufbar bleiben.

Im Projekt BIS-Ares fiel mir die Aufgabe zu, das Datenmodell und die
Datenbankarchitektur zu entwerfen. Reizvoll fand ich, dass meine Arbeit
die Grundlage für beide anderen Teile bildete: Weder Frontend noch Backend
funktionieren ohne ein sauberes, durchdachtes Modell. Mit PostgreSQL
konnte ich ein mächtiges, bewährtes System einsetzen -- und gleichzeitig
lernen, wie sehr gute Modellierung später Arbeit und Fehler spart.

\emph{Negin Sadeghi}
