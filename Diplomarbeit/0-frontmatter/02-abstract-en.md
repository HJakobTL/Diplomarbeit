# Abstract {.unnumbered .unlisted}

This thesis is part of the joint project **BIS-Ares**, a citizen
information system developed for the City of Vienna municipal authority
(MA~35). The overall project aims to replace fragmented, manual case
documentation with a modern, fully digital web application that lets
citizen-service staff record, manage and evaluate cases centrally.

The project was carried out by three students of the HTBLVA Vienna V,
Spengergasse (class 6AKIF) during the school year 2025/26 and was
divided into three individually scoped theses:

- **Frontend & UI design** (Samaneh Madani) -- conception and
  implementation of a responsive, accessible web interface using
  Angular and Tailwind CSS;
- **Backend & REST API** (Maria Binder) -- server-side implementation of
  the business logic and a versionable REST interface built with Node.js
  and Express;
- **Database & data modelling** (Negin Sadeghi) -- design and
  implementation of the relational data model using PostgreSQL.

The work did not participate in any competition. The thesis can be
inspected at HTBLVA Vienna V, Spengergasse.
